# <Icon iconContent contentColor contentHeightInRem darkMode />

Icon component. Gets icon content from Icon Registry. Connected to Redux store.

PropTypes:
Name Type Required Description Default
iconContent string Yes Which SVG should be used. E.g., "Home".
contentColor string No Fill color for SVG. Will be used to find color from Style service. E.g., "ux-pink". 'ux-base-text'
contentHeightInRem string Yes Height of icon, measured in rem. E.g., "4".
darkMode bool No From state store. Whether user prefers dark mode. E.g., "true". false
